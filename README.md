# Numerical Computing Solutions

Bu depo, muhendislik calismalarim kapsaminda gelistirdigim sayisal analiz algoritmalari ve temel programlama cozumlerini icermektedir. Projeler C# ve Python dilleri kullanilarak kodlanmistir.

## Proje Icerigi

### 1. Root-Finding Methods
Matematiksel fonksiyonlarin koklerini bulmak icin kullanilan iterative yontemleri icerir.
- Bisection ve Euler yaklasimlari (C#)
- Newton-Raphson algoritmasi (Python)
- Jacobi ve Gauss-Seidel sistem cozuculeri (Python)
- Polinom Regresyon analizi (C#)

### 2. Linear Algebra Solvers
Matris operasyonlari ve lineer denklem sistemlerinin cozumu uzerine calismalar.
- Gauss-Jordan eliminasyon yontemi (C#)
- Dinamik katsayilar matrisi olusturucu (Python)

### 3. Calculus Simulations
Turev ve integral tabanli sayisal hesaplamalar.
- Euler yontemi ile integral hesabi ve e sayisi yaklasimi (Python)

### 4. Logic Challenges
- Asal sayi eleme ve dongu mantigi uzerine kurulmus algoritma calismasi (C#)

## Teknik Detaylar
Kodlar Visual Studio ve VS Code ortamlarinda test edilmistir. Sayisal yontemlerin bilgisayar bilimlerindeki uygulamalarini gostermek amaciyla hazirlanmistir.
